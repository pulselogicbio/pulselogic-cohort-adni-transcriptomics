# Data Access -- Alzheimer's Disease - Transcriptomics (ADNI U219 microarray)

## Data Source

Expression and survival data for this cohort were obtained from Alzheimer's Disease Neuroimaging Initiative (ADNI). Access requires registration at https://adni.loni.usc.edu.

---

## Accessing This Data

1. Register at https://adni.loni.usc.edu
2. Apply for data access (IDA portal)
3. Search for cohort: **ADNI U219 microarray**
4. Download relevant expression/genomic and clinical files
